How it works

  1. Authenticates with Multilogin and starts the configured browser profile.
  2. Collects creator usernames from Instagram explore / location hashtags.
  3. Screenshots each creator's profile and top posts.
  4. Sends screenshots to Claude Vision for scoring and pitch generation.
  5. Saves each lead to the database and renders a PDF audit.
The scout is designed for Windows/Multilogin. If it cannot start on this machine, the error will appear in the run log below.
